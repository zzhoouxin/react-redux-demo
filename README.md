## 技术支持
- 前端：`react`、`redux`


## 运行步骤
1.demo1 原生版   ==>   cd/views/redux-html   ==>点击index.html
2.demo2  react+redux
 ==> cd/views/react_redux
 ==> npm install
 ==> npm run build
 ==> npm run start
  

## 项目目录结构

```
项目

    |__ views     //视图
        |__redux-html    //原生版
            |__index.html   //script reudx.js
        |__react_reudx    //react+redux 实现的demo
            |__ node_modules 依赖
            |__build    //编译出来的js
            |__public    //模板
            |__src  //代码位置
                |__components  //组件
                |__ reducres  //数据源处理
                |__index.js   //首页


```
